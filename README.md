# ML Ethics Checklist

Android-приложение на Java для локального аудита этических рисков ML-проекта.

## Что реализовано

- проекты ML-аудита;
- история проверок для одного проекта;
- чек-лист по категориям: данные, предвзятость, качество модели, интерпретируемость, приватность, мониторинг;
- ответы по пунктам: да, нет, частично, не применимо;
- комментарии аудитора и отметка “добавить в проблемы”;
- автоматическое формирование списка проблем и рекомендаций;
- статусы проблем: новая, в работе, исправлена, принято как ограничение;
- итоговая оценка риска и прогресса;
- текстовый отчёт по активному аудиту;
- локальное хранение данных в SQLite.

## Технически

- Java;
- стандартные `Activity` и `Fragment` без AndroidX;
- `SQLiteOpenHelper`;
- `ListView`, `Spinner`, `AlertDialog`;
- без внешних зависимостей.

Проект можно открыть в Android Studio как обычный Gradle Android-проект.
